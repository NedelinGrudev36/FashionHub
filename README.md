# 🖤 FashionHub — Modern Streetwear Website

FashionHub е модерен, изчистен и стилен streetwear сайт, изграден изцяло с **HTML и CSS**.  
Проектът показва напълно готов front-end дизайн на модерен онлайн магазин за мъжка мода.

Сайтът включва навигация, категории, хедър секция, продуктови блокове, FAQ страница и модерен минималистичен футър.

---

## 🔥 Основни функции

### 🔹 Навигация
- Изчистено и модерно navigation меню  
- Линкове към: **Home, Shop, FAQ, Contact**  
- Под линк „Shop“ води към страница/секция с всички категории

### 🔹 Header / Hero Секция
- Голям банер с модерна типография  
- Представя идеята на бранда FashionHub  
- Изцяло в streetwear стил

### 🔹 Категории (Shop by Category)
Основните категории на FashionHub са:
- **Tops**
- **Bottoms**
- **Outerwear**
- **Sneakers**
- **Accessories**

Всяка категория има собствена представителна снимка в еднакъв стил.

### 🔹 Featured / New Collections
- Специална секция за представени продукти  
- Подредба в grid стил  
- Всяка снимка е квадратна и еднаква по размер

### 🔹 Modern Footer
- Лого / Име на бранда  
- Shop линкове  
- Help секция  
- Social Media икони  
- Copyright текст  
- Изцяло тъмен urban дизайн
